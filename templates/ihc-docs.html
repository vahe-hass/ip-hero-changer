Once you've meticulously crafted your homepage using the Elementor or WordPress editor, it's crucial to establish it as the designated homepage.

For Elementor, refer to the accompanying image, and navigate to the button widget settings. Insert your preferred ID name in the provided field, taking care not to include the ID in the button itself.

If you're using the WordPress editor, access the buttons block and integrate the desired ID name into the HTML anchor within the block settings. Ensure that you promptly apply and save the modifications to reflect the updated changes.

To initiate the IP Hero Changer plugin's functionalities and gather data, the initial step involves saving your preferences through the form. Proceed to the bottom of the page to configure the necessary options. Choose the region, country, and state for which you want IP Hero Changer to modify the button color when visitors attempt to access your homepage from that specific state.

Once these selections are made, input the HEX color for the button you wish to subject to A/B testing. This Hex color serves as your B option in the A/B testing scenario, with the initial design designated as your A option.

To conclude the setup, input the same ID that you added in the first step, starting with the "#" character (ensure the inclusion of "#" to facilitate successful form submission). Finally, click the "Save Settings" button to apply and store your configured preferences.

To eliminate and reset your homepage A/B testing and filters, simply click the "Delete All Data" button. This action will erase all configured settings and data. It is advisable to take a backup or make a note of your data before proceeding, as the IP Hero Changer does not automatically create backups.
